Что такое DNS (Domain Name System) 
в чём разница протоколов HTTP/HTTPS и их версий

DNS - перевод/обработка буквенного обозначения сайта в его цифровое значение (IP адрес) и обратно

HTTP - протокол взаимодействия между клиентом и сервером
Версии HTTP протокола 0.9, 1.0, 1.1, 2.0
Преимущества протокола 2.0 к 1.1: 
- версия 2.0 использует бинарный формат, что уменьшает объем передаваемой информации и увеличивает производительность. 
Версия 1.1 использует текстовый формат
- версия 2.0 все запросы и ответы отправляет и принимает одномоменто/параллельно, используется одно TCP соединение для всех запросов и ответов.
Версия 1.1 передает запросы последовательно. И каждый запрос/каждый ответ проходит по отдельному TCP соединению. Что увеличивает время передачи информации.

HTTPS - применяет способы защиты информации и является более безопасным протоколом отоносительно HTTP.
Имеют разные порты: 80 или 8080 для HTTP и 443 для HTTPS

HTTPS имеет протоколы защиты SSL (не поддерживается с 2015 года) и TLS.

В связи с наличием системы защиты при передаче данных в HTTPS имеет более сложное взаимодействие между клиентом и сервером относительно HTTP.
